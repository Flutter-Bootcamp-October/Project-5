# Project-5

## project overview:
The CV application is designed to facilitate the systematic organization of your resume, enabling users to securely store their professional information within the application framework. This application affords users the capability to modify their profile details, including contact information, and to incorporate pertinent elements such as projects, skills, educational background, and social media profiles.
